# Mihomo Kit for Keenetic + Entware

Простой комплект для работы с Mihomo на Keenetic.

## Возможности

- Mihomo Core
- Zashboard (порт 9090)
- Mihomo SubManager (порт 9091)
- Mihomo Redirect для прозрачного TCP-прокси
- Автоматическое применение подписок
- Встроенный набор правил GEOSITE

## Установка

```sh
sh -c "$(wget -qO- https://raw.githubusercontent.com/air404air/mihomo-kit/main/install.sh)"
